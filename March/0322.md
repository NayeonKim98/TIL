# 기사 리뷰

- 링크 : https://www.aitimes.com/news/articleView.html?idxno=168948

- 내용 : LLM을 통해서가 아닌, 진짜 '개'처럼 현실세계를 통해 학습하는 로봇이 개발되었다. 데이터를 미리 넣어주는 것이 아닌, 경험하면서 스스로 배우는 방식이다.

- 디지털 신경계(digital nervous system) : 감각신호(sensory signals)를 분석하고, 그걸 기반으로 스스로 행동을 개선하는 구조. 즉, 사람이나 동물이 신경계로 학습하는 방식과 유사.

- 원래 로봇과의 차이점 : 훈련사가 필요하지만 대형 데이터셋/사전훈련/시뮬레이션들은 필요 없음. 스스로 학습

- 활용가능성 : 우주탐사, 재난 대응, 극한 환경에서 도움. 미리 학습시키기 어려운 환경에서 스스로 판단하고 움직일 수 있음.
